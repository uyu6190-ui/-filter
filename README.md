# perl rabbit

写真・動画に「透明感フィルター」をかける、ブラウザだけで動くWebアプリです。

## できること

- 写真または動画を追加
- ワンタップで透明感フィルターを適用
- フィルターの強さを調整
- Before / After を切り替え
- 加工後の画像・動画を保存

すべて端末内で処理され、ファイルは外部へ送信されません。

## 起動方法

依存関係はありません。`index.html` を直接開くか、簡易サーバーで確認できます。

```bash
python3 -m http.server 8001
```

ブラウザで `http://localhost:8001/` を開いてください。
